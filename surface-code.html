About Surface Code

The surface code (Kitaev 2003) is the leading candidate for fault-tolerant quantum computing, offering a threshold of pth ≈ 1%.

Data qubits sit on a 2D lattice. Stabilizers Av = ⊗X and Bp = ⊗Z are measured via ancilla qubits — any violated stabilizer (±1 eigenvalue) signals a nearby error.

A minimum-weight perfect matching (MWPM) decoder pairs syndromes and applies corrections. Below threshold, logical error rate drops exponentially with code distance d.

pL ≈ A(p/pth)⌊(d+1)/2⌋
