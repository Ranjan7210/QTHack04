Quantum Algorithms

The two most important quantum algorithms — visualized step by step.

🔍 Grover's Search Algorithm

Find a needle in a haystack — quadratically faster than any classical search.

1

Superposition

Put all N items into equal superposition using Hadamard gates. Each item has probability 1/N.

2

Oracle

Mark the target item by flipping its amplitude sign. The oracle "knows" the answer but only reveals it quantum-mechanically.

3

Amplification

Apply the "diffusion operator" — reflect amplitudes about the mean. The target gets boosted, others shrink.

4

Repeat √N times

After ~√N iterations, the target amplitude is nearly 1. Measure to find it with high probability.

🔐 Shor's Factoring Algorithm

Factor large numbers in polynomial time — the reason RSA is threatened.

1

Choose Random a

Pick a random number a < N. If gcd(a, N) > 1, you already found a factor. Otherwise continue.

2

Find the Period r

Use a quantum computer to find the period r of f(x) = ax mod N. This is the hard part classically; quantum does it efficiently using QFT.

3

Extract Factors

If r is even, compute gcd(ar/2 ± 1, N). These are likely non-trivial factors of N.
