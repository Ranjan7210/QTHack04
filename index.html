How It Works

QuantumViz runs a real physics engine in your browser — no server, no install.

01
Choose a Module

Select from curated experiments spanning foundations to advanced algorithms. Each module is self-contained and immediately interactive.

02
Manipulate the Physics

Drag Bloch vectors, inject quantum errors, tune Hamiltonians with sliders, or braid anyons with your mouse. Every control maps to a real quantum operation.

03
See the Math Update

Toggle the Formalism layer to see live bra-ket notation, stabilizer equations, and expectation values respond to every physical change in real time.

04
Build Intuition

Predict outcomes with Predictive Challenge mode. Misconception detection surfaces targeted micro-lessons when common errors (e.g. ignoring relative phase) occur.

Three Layers of Understanding

Progressive disclosure keeps beginners engaged and experts satisfied. Switch layers at any time — the simulation doesn't pause.

🌊
Layer 1 · Intuition

Probability clouds, spin arrows, entanglement threads. Phase encodes to hue; amplitude encodes to brightness. Zero equations required.

∑
Layer 2 · Formalism

Bra-ket notation overlays on every element. Live Hamiltonian, stabilizer generators Av, Bp, and expectation values ⟨A⟩ alongside the animation.

⌨
Layer 3 · Computation

Step-by-step computation log. Numerical methods exposed — matrix exponentiation, MWPM matching, Monte Carlo shot counts. Export pseudocode to Qiskit.

The qubit lives in a superposition of |0⟩ and |1⟩ until measured.
Bar height = probability of each outcome.

|ψ⟩ = α|0⟩ + β|1⟩
α = cos(θ/2)
β = eiφ sin(θ/2)
|α|² + |β|² = 1  (normalization)
Prob(0) = ⟨ψ|P₀|ψ⟩ = |α|²
from qiskit import QuantumCircuit
import numpy as np

qc = QuantumCircuit(1, 1)
qc.ry(2 * theta, 0)   # set θ
qc.rz(phi,        0)   # set φ
qc.measure(0, 0)
# Born rule: P(0) = cos²(θ/2)
